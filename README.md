# antivirus-server
Secure Docker Based Antivirus File Scanning Server

A Linux-based antivirus server that securely scans files for potential threats using SSH-based file transfer and Dockerized analysis services.

## Tech Stack
- Python
- Docker
- Linux
- Shell Scripting
- SSH

## Features
- Secure file upload via SSH
- Real-time file scanning
- Containerized analysis for isolation and reliability
- Low-latency response handling

## How It Works
1. Client uploads a file using SSH
2. File is passed to a Docker container for analysis
3. Server returns scan results to the client
